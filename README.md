## 📚 Learning Objectives

  - **Deconstruct the anatomy of an agent:** Dive into how models, harnesses, and tool integrations operate under the hood, and what that means for production-grade deployment on AWS.
  - **Engineer custom agent logic:** Use the Strands Agents SDK to build extensible, modular agents, focusing on real-world orchestration patterns that scale in AWS environments.
  - **Wire up model providers:** Connect Amazon Bedrock, Anthropic, and OpenAI, and run local inference with Ollama. Benchmark, swap, and optimize foundation models for your specific AWS workloads.
  - **Integrate external tools and data sources:** Leverage the Model Context Protocol (MCP) to unlock agentic workflows that interact directly with your AWS environment.

## 🛠️ Tech Stack

  - **Framework:** Strands Agents SDK (Python/TypeScript)
  - **Models:** Amazon Bedrock (Nova, Claude), Ollama (local inference)
  - **Protocols**: Model Context Protocol (MCP)
  - **Development:** Python, TypeScript
